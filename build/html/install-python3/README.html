How to install python3¶

Simplified guide on how to install python3 on your computer.

Windows¶

  1. Go to: https://www.python.org/downloads/
  2. Click button: Download Python 3.#.#
  1. Open installer Python 3.#.#.exe
  2. Check Add Python to PATH
  1. Click Customize installation
  2. On next screen check all boxes and click next
  3. In Advanced options make sure you have checked following: 1. Add Python to environment variables 2. Change the installation location to: C:\Users\<Your-UserName>\Python\Python36-XX (XX is a version number)

8. Click install 9 You are finished with this lab!

Mac¶

  1. Go to: https://www.python.org/downloads/
  2. Click button: Download Python 3.#.#
  3. Open pkg installer and follow the instructions.

Linux¶

Depends your distribution. You can always install python3 from source. Which can be found at https://www.python.org/downloads/.

Ubuntu (16.10+): apt install python3.6 Fedora 26+: should include python 3.6, previous versions: dnf install python36 Centos 7: follow this guide https://janikarhunen.fi/how-to-install-python-3-6-1-on-centos-7.html

Try your python¶

Once your python is installed you can try it:

  1. Open the terminal emulator.

    • Mac OSx: hit [CMD]+[Space] and type terminal.app
    • Windows: hit [WIN-key]+[R] and type powershell (or you can search for powershell in start menu.

  2. Type command:

    python -V
    • Output should give you version of your python 3.
    Python 3.6.X
    • If your system replies with Python 2.7.X use the command:
    python3 -V
    • MacLinux TIP: if you do not want to type python3 every time add it as an alias for your user. To make alias permanent check this site: https://jonsuh.com/blog/bash-command-line-shortcuts/
    alias python="python3"

  3. Start the python shell with command:

    python
    • Command above will start python shell
    Python 3.6.5 (v3.6.5:f59c0932b4, Mar 28 2018, 03:03:55)
[GCC 4.2.1 (Apple Inc. build 5666) (dot 3)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>>

  4. Type the code below and hit [Enter] to display the Zen of python.

    import this

  5. To exit the shell type:

    exit()

Great! You have successfully installed python.
